Conditional Statements:-

1) if statement 

2) if else statement 

3) else if statement or else if ladder

syntax:-

	if(condition)
	{
		statement;
	}
	else if(condition)
	{
		statement;
	}
	else if(condition)
	{
		statement;

	}
	else
	{
		statement;

	}


4) Nested if statement:-

syntax:- 	

	if(condition)
	{
		if(condition)
		{
			statement;
		}

		statement;
		else
		{
			
		}

	}
	else
	{

	}

5) Switch Statement:- 

syntax:-


		switch(varname)
		{
			case 1:
			statement;
			break;
		
			case 2:
			statement;
			break;

			case 3:
			statement;
			break;


			default:
			statement

		}
